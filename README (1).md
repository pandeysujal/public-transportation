# 🚆 Smart Cities & Future Transportation Quiz

A Python-based command-line quiz app that tests your knowledge of smart city technology, autonomous vehicles, IoT in transportation, and the future of urban mobility.

Built as a collaborative GitHub project for CIS coursework.

**Authors:** Sujal, Bikas & Bryce

---

## 📋 Topics Covered

- Adaptive traffic signal systems
- Autonomous vehicle sensors (LiDAR)
- Vehicle-to-Vehicle (V2V) communication
- Mobility as a Service (MaaS)
- IoT in transportation infrastructure
- Electric public transit
- Congestion pricing
- Digital twins in urban planning
- Hyperloop technology
- Smart city global examples

---

## 🚀 How to Run

Make sure you have **Python 3** installed.

```bash
# Clone the repository
git clone https://github.com/your-username/smart-city-transport-quiz.git

# Navigate into the folder
cd smart-city-transport-quiz

# Run the quiz
python quiz.py
```

---

## 🎮 How It Works

1. Launch the quiz in your terminal
2. Answer 10 multiple choice questions (A / B / C / D)
3. Get instant feedback + explanation after each question
4. See your final score and performance rating at the end
